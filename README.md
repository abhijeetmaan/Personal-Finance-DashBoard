# 💰 Finova – Personal Finance Dashboard

A modern, full-stack **Personal Finance Management App** built with MERN stack that helps users track expenses, manage accounts, monitor budgets, and gain financial insights with a clean and premium UI.

---

## 🌐 Live Demo

* 🔗 Frontend: https://personal-finance-dash-board.vercel.app/
* ⚙️ Backend: https://personal-finance-dashboard-6-1pn7.onrender.com

---

## ✨ Features

### 💸 Finance Management

* Add / Edit / Delete transactions
* Track income & expenses
* Category-based spending

### 🏦 Accounts & Cards

* Manage multiple bank accounts
* Credit card tracking with billing cycle
* Real-time balance updates

### 📊 Analytics & Insights

* Net Worth calculation (Assets - Liabilities)
* Monthly trends & charts
* Category-wise spending analysis
* AI-like financial insights

### 🎯 Budget System

* Set monthly budgets
* Category-wise budget tracking
* Smart alerts (over budget warning)

### ⚡ Advanced Features

* Double-entry accounting (ledger system)
* Real-time updates (WebSockets)
* Toast notifications & modals
* Mobile-first UI (like GPay)

---

## 🎨 UI/UX Highlights

* Premium fintech-style design (Stripe-inspired)

## 🚀 Installation (Local Setup)

### 1. Clone repo

git clone https://github.com/abhijeetmaan/Personal-Finance-DashBoard.git
cd Personal-Finance-Dashboard

---

### 2. Setup Backend

cd backend
npm install
npm run dev

---

### 3. Setup Frontend

cd frontend
npm install
npm run dev
